# gestion_programme

nous sommes une équipe de développeurs qui a développé ce projet pour gérer les programmes d'un établissement.

le but de ce projet est de permettre la gestion des programmes d'un établissement.
nous avons choisi en utilisant php pour le backend et html, css et js pour le frontend.

nous avons choisi de mettre en place un systeme de gestion de base de données pour stocker les informations des programmes.
voici les fonctionnalités du projet :
- la gestion des utilisateurs
- la gestion des programmes
- la gestion des matieres
- la gestion des notes
- la gestion des absences
- la gestion des bulletins
- le suivi des l'apprenants
- l'organisation des cours
- la gestion des examens

sur le plan technique, nous avons la structure suivante :
- le backend est en php
- le frontend est en html, css et js
- la base de données est en mysql

nous utilisons le mvc pour structurer le projet.
- le model est en php
- le controller est en php
- le view est en html, css et js
